# VERA v2 — Public Deposit (Deterministic Action Governance)

This folder is a **reference artifact**, not a product.

It demonstrates:
- Proposal vs Commit separation
- Drift detection (REQUIRE_RECOMMIT)
- Fail-closed defaults (REFUSE)
- Stable reason codes + stable human output (Coat)
- Audit-safe ledger events
- Multi-domain modularity (Hats) with explicit selection (Router)

## What’s included

- `public/README_PUBLIC.md` — overview
- `public/ONE_PAGER.md` — investor summary
- `public/APPLE_GAP_MAP_MONTH12.md` — neutral platform gap map
- `public/APPLE_PITCH_OUTLINE.md` — Apple-facing outline
- `public/DEMO_SCRIPT_VERBAL.md` — what to say while demo runs
- `public/INVESTOR_NARRATIVE.md` — longer narrative
