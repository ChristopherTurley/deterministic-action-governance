# Security Policy

## Scope
This repository is a governance-only reference artifact.

It must not contain:
- secrets, API keys, or tokens
- credentials or personal data
- broker integrations that execute trades
- automation that performs side effects

## Reporting
If you believe you have found a security issue, open a GitHub issue with minimal detail
or contact the maintainer privately.

## Guarantees
- Deterministic behavior enforced by tests
- No background behavior
- No side effects without explicit operator commit
