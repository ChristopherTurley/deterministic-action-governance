## What This Repository Is

This repository is **not** a product runtime and **not** an AI agent.

It is a **governance reference artifact** that demonstrates how an AI system can be:
- useful
- deterministic
- auditable
- fail-closed by default

VERA exists to answer a narrow but important question:

> How do we allow AI to assist humans **without** allowing it to act silently, unpredictably, or without accountability?

Everything here is structured to make boundaries explicit.

---

## What May Feel Unusual

Some things in this repository are intentionally different from typical AI projects:

- **Refusals are success states**, not failures
- **No background execution** exists anywhere in this system
- **All actions are gated, logged, and reviewable**
- **Nothing here performs automation**, even in demos

This is by design.
VERA prioritizes governance correctness over capability breadth.
